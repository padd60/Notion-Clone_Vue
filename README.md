# Vue3 Webpack Template

- Webpack + Babel + PostCSS(SCSS)
- Vue3 + Vuex + Vue-Router
- ESLint

## 클론 후 .git 삭제하기

```bash
# macOS 
rm -rf .git

# Windows
rmdir /s .git 
```

## 프로젝트 시작 전 패키지 설치

```bash
npm i
```

## 개발 서버 시작

```bash
npm run dev
```
